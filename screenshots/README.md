## Tips

- PNG for UI screenshots, JPG for the hero photo
- Crop screenshots tight (no browser chrome)
- Keep file sizes under ~400KB if possible — [tinypng.com](https://tinypng.com) helps

After dropping or replacing files, just reload. No build step.
